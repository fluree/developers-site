When the user clicks on the _Data Set_ tab on the left menu, they will be directed to the main Data Set page. This page will include all the datasets that the user has access to, as well as some information about these datasets including:

1. Data Set name

3. Data Set description

5. Data Source of the Data Set

7. Who created the Data Set

9. The last time the Data Set was modified

11. The Classification Tag of the Data Set
